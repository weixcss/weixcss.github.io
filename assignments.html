Assignment 3: SVD Preprocessing on MNIST with Logistic Regression

This assignment involves loading the MNIST dataset to normalize it. Performed SVD and reduced the dimensions of the data. Trained a logistic regression model on the original and SVD-reduced data. Measured and compared the training time and accuracy of the model with varying SVD components. Results shown in a plot with analysis on how SVD impacts the performance and efficiency of the model.

Assignment 6: Linear Regression

Exploration: Impact of changing parameters on linear regression. The goal is to create an interactive webpage to demonstrate how modifying these parameters affects regression results, especially when there is no actual relationship between X and Y. By tweaking these settings, we'll observe how randomness can influence the slope and intercept in a regression model.
